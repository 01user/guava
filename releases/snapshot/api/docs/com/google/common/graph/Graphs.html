        public static <N,E> Set<E> parallelEdges(Network<N,E> graph,
                         Object edge)
        Returns an unmodifiable view of edges that are parallel to edge, i.e. the set of edges that connect the same nodes in the same direction (if any). An edge is not parallel to itself.
        Throws:
        IllegalArgumentException - if edge is not present in graph

      • inducedSubgraph

        public static <N,E> MutableNetwork<N,E> inducedSubgraph(Network<N,E> graph,
                                        Iterable<? extends N> nodes)
        Returns an induced subgraph of graph. This subgraph is a new graph that contains all of the nodes in nodes, and all of the edges from graph for which the edge's incident nodes are both contained by nodes.
        Throws:
        IllegalArgumentException - if any element in nodes is not a node in the graph
