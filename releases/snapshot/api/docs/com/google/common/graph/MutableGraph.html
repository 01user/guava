      • addEdge

        boolean addEdge(N node1,
              N node2)
        Adds an (implicit) edge to this graph connecting node1 to node2 (optional operation).

        Behavior if node1 and node2 are not already elements of the graph is unspecified. Suggested behaviors include (a) silently adding node1 and node2 to the graph or (b) throwing IllegalArgumentException.

        Currently, this type does not support parallel edges. addEdge(node1, node2) will simply return false on any future calls with the same arguments (analogous to the behavior of Network.addEdge(e, node1, node2)). A hypothetical instance that supported parallel edges would add a new edge between node1 and node2 for every call to addEdge(node1, node2), and return true every time.

        Returns:
        true iff the graph was modified as a result of this call
        Throws:
        UnsupportedOperationException - if the add operation is not supported by this graph
