Class Endpoints<N>

  • java.lang.Object
    • java.util.AbstractCollection<N>
      • com.google.common.graph.Endpoints<N>
  • All Implemented Interfaces:
    Iterable<N>, Collection<N>

    @Beta
public abstract class Endpoints<N>
extends AbstractCollection<N>
    An immutable Collection to represent the two (possibly equal, in the case of a self-loop) endpoints of an edge in a graph.

    If an Endpoints is directed, it is an ordered pair of nodes (source and target). Otherwise, it is an unordered pair of nodes that can be accessed through the iterator.
