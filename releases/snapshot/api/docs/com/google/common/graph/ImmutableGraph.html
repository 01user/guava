com.google.common.graph

Class ImmutableGraph<N,E>

  • java.lang.Object
    • com.google.common.graph.AbstractGraph<N,E>
      • com.google.common.graph.ImmutableGraph<N,E>
  • Type Parameters:
    N - Node parameter type
    E - Edge parameter type
    All Implemented Interfaces:
    Graph<N,E>

    public final class ImmutableGraph<N,E>
extends AbstractGraph<N,E>
    A Graph whose contents will never change. Instances of this class should be obtained with copyOf(Graph).

    The time complexity of edgesConnecting(node1, node2) is O(min(outD_node1, inD_node2)).

    Author:
    James Sexton, Joshua O'Madadhain, Omar Darwish
