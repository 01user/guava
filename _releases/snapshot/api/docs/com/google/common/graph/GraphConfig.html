com.google.common.graph

Class GraphConfig

  • java.lang.Object
    • com.google.common.graph.GraphConfig

  • @Beta
public final class GraphConfig
extends Object
    A class for configuring different types of graphs.

    Graphs.config() should be used to get an instance of this class.

    Currently, this class supports the following graph configurations (all combinations of these properties are valid unless stated otherwise):

    • Multigraphs.
    • Expected number of nodes/edges.
    • Self-loop edges.

    Default graph configuration:

    • Self-loop edges are allowed.
    • It is not a multigraph: parallel edges (multiple edges directed from n1 to n2, or between them in case of undirected graphs) are not allowed.
    • In case of directed graphs, anti-parallel edges (same incident nodes but in opposite direction, e.g. (n1, n2) and (n2, n1)) are allowed.
    • Nodes and edges are not sorted.

    GraphConfig instances are thread-safe immutable, and are therefore safe to store as static final constants.

      • compatibleWith

        public boolean compatibleWith(GraphConfig that)
        Returns true iff the configuration defined by this object is compatible with the configuration defined by that. Intuitively, this checks for structural properties and ignores non-structural properties. Specifically, this method checks whether the configurations have compatible support for:
        • parallel edges
        • self-loops

        It does not compare expected values for numbers of edges or nodes, and it is not equivalent to Object.equals.
